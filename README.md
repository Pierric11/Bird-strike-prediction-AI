# Bird-strike-prediction-AI
Teams member: Pierric Vernet, Andrea Remy, Milan Eliayan, Antonin Lo

Bird strike in Miami INTL:

The goal of the project is to predict when a bird strike might occur at Miami International Airport on a given day.
The code returns the times of day when a bird strike might occur, the weather conditions, the flight phase
and the most likely aircraft.
